# Meme Generator

A lightweight React application for creating memes with custom text and downloading the final image. Built with Vite, React, and CSS Modules.

Fully optimized with perfect Lighthouse scores and installable as a Progressive Web App (PWA).

## Features

- Generate random meme images from the [ImgFlip API](https://imgflip.com/memegenerator)
- Add custom top and bottom text
- Download the generated meme as a PNG
- Installable as a PWA
- Achieves a perfect 100 in all Lighthouse categories
- Styled using CSS Modules
- More suitable for desktop/web than mobile

## Tech Stack

- React
- Vite
- CSS Modules
- Canvas API
- ImgFlip Meme API
